# 二人対戦専用ピンボール

## 制作LPの説明
- 昔懐かしのピンボールゲームです
- 二人対戦で遊ぶことを想定してます（オフライン専用）

## 工夫した点・こだわった点
- スムーズにボールやバーが動かせるように処理を工夫しました
- 屈折するお邪魔バーを一定範囲で動かせるようにすることや、位置の保存、読み込み、特定の角度には変化しないようにするなど、このあたりの設定はゲームとしてストレスがなくなるように工夫しました

## 次回トライしたこと（または機能）
- ローカルストレージをうまいこと利用した機能や遊び方がなかなか思いつかなかったので、データを保存することならではといえる遊び方を考えたいです
- リアルタイムのタイミングシビアなゲームじゃないもので、オンライン対戦みたいなことをやってみたいです
